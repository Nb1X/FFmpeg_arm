# FFmpeg for ARM64 Debian Bookworm

FFmpeg, on Debian Bookworm for ARM, is only available on older versions that are not compatible with software like OBS.

So I compiled FFmpeg 7.0 on a Raspberry Pi 5 running Debian Bookworm.
You can use it on any other ARM device running a 64-bit OS.

This project is made for my other project: https://github.com/Nb1X/obs-arm64
